# Personal AI Employee
Built for Panaversity AI Employee Hackathon.

## Current Tier: Bronze
## Stack: Claude Desktop + MCP + Python + Obsidian

## Quick Start
1. Install Python 3.11+, Node.js 20, Claude Desktop, Obsidian
2. cd ai-employee-core
3. uv venv
4. .venv\Scripts\Activate.ps1
5. uv pip install python-dotenv watchdog
6. Configure Claude MCP (see setup docs)
7. Open AI_Employee_Vault in Obsidian
8. python watchers\filesystem_watcher.py
